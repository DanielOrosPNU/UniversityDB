<h2>Як використати?</h2>
Завантажте за допомогою
<pre>
  git clone https://github.com/DanielOrosPNU/UniversityDB
</pre>
Створіть базу даних
<pre>
  create database TestDB
</pre>
Почніть використовувати скрипти по черзі.
<pre>
  1. Tables.sql
  2. ForeignKeys.sql
  3. Insert.sql
  4. Select.sql
  5. View.sql
  6. HistoryTables.sql
  7. Get_Set_StoredProcedure.sql
  8. Business_StoredProcedure.sql
</pre>
